# Medical Centre

![Brand](brand.png)

It's HTML template for Wordpress custom theme.

## Getting start

### Installing

To start work with this project you need to install modules, use

```
npm instal
```

to install all dependencies.

### Convert scss to css

Run this command in root folder:

```
npm run start
```
